# sarveshkapre.github.io

Personal site + blog built with Astro (Astro Cactus template).

## Writing

- Add posts in `src/content/post/*.md`
- Blog index: `/posts/`
- Tags: `/tags/`
- RSS feed: `/rss.xml`

## Dev

- `npm install`
- `npm run dev`

## Build

- `npm run build` (outputs to `dist/`, also builds Pagefind search)
- `npm run preview`

## Deploy

GitHub Pages deploys via `.github/workflows/deploy.yml` on pushes to `main`.
