# sarveshkapre.github.io

Personal site + blog built with Astro.

## Writing

- Add posts in `src/content/blog/*.md`
- Blog index: `/blog/`
- RSS feed: `/rss.xml`

## Dev

- `npm install`
- `npm run dev`

## Build

- `npm run build` (outputs to `dist/`)
- `npm run preview`

## Deploy

GitHub Pages deploys via `.github/workflows/deploy.yml` on pushes to `main`.
